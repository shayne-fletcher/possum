Expand description

Apply file selection: keep files matching any include glob (or all when include is empty), then drop any matching an exclude glob. Only in the default case (no explicit include) do we also drop the *.pt/*.bin duplicates of safetensors weights; an explicit include is taken as the caller knowing exactly what they want.
